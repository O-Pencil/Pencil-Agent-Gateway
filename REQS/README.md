# Pencil-Agent-Gateway Requirements Registry
> P1 | REQS/ — Gateway feature backlog, structured as metadata cards for team evaluation
> Each file = one requirement. Naming convention: `REQ-<id>-<slug>.md`

---

## How to use this directory

- Each `.md` file is one self-contained requirement card
- Format: `[WHO]/[FROM]/[TO]/[HERE]` DIP header + structured body
- Requirements are **consumed by**: Gateway maintainers, integration reviewers, PM
- Sort order: priority within each file, no sorting across files needed
